Adding Your Own Content

This static archive is a read-only showcase. On the live Speakize site, you can add your own study material using the features described below.

Add YouTube

Paste a YouTube link and Speakize will fetch the video's subtitles automatically, turning them into study material:

  • Each subtitle line becomes a clickable phrase
  • Audio from the video is linked back to each phrase
  • Create flashcards directly from subtitle lines
